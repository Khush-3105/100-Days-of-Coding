# Problem: Pow(x, n)

Platform: Leetcode

Difficulty: Easy

Problem Link: https://leetcode.com/problems/powx-n/

## Problem Statement:

Implement pow(x, n), which calculates x raised to the power n (i.e., xn).

Example 1:

    Input: x = 2.00000, n = 10
    Output: 1024.00000

## Solution (In Java):
    
    class Solution {
        public double myPow(double x, int n) {

            return Math.pow(x,n);
        }
    }
